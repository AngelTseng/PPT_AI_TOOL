# PPT AI Tool

This tool generates PowerPoint using AI and a template.

Requirements
------------
1. Windows
2. Python 3.10+
3. Microsoft PowerPoint installed
4. Python module:
  streamlit
  pywin32
  python-pptx
  pydantic
  openai
  tqdm

Installation
------------
Double click:

install.bat

Run
------------
Double click:

run_ui.bat

The UI will open in your browser.

Default address:
http://localhost:8501
